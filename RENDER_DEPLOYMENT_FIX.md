## 🔍 Troubleshooting Common Issues

### Issue 1: Service Won't Start
**Symptoms**: Logs show errors during startup
**Solutions**:
1. Check DATABASE_URL is properly set
2. Verify all migrations ran successfully
3. Check SECRET_KEY is set
4. Review build logs for pip install errors

### Issue 2: Database Connection Failed
**Symptoms**: `could not connect to server` or `database does not exist`
**Solutions**:
1. Verify database `tdms-db` exists and is available
2. Check DATABASE_URL environment variable
3. Ensure service is linked to database in Render
4. Try recreating the database connection

### Issue 3: CORS Still Blocking Requests
**Symptoms**: Browser console shows CORS errors
**Solutions**:
1. Verify CORS_ALLOWED_ORIGINS includes `https://tdms-one.vercel.app`
2. Clear browser cache
3. Check service redeployed after changes
4. Use browser DevTools → Network tab to inspect headers

### Issue 4: 502/503 Errors Persist
**Symptoms**: Backend returns 502 or 503
**Solutions**:
1. Check service logs for crash errors
2. Increase memory/resources if available
3. Check if gunicorn workers are configured correctly
4. Verify WSGI application path is correct
